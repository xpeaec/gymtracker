Cara deploy ke GitHub Pages (gratis & online terus)
File HTML ini sudah self-contained — semua data tersimpan di file JSON yang kamu export/import. Tidak perlu database, tidak perlu server. GitHub Pages hosting-nya gratis selamanya.
Langkah 1 — Buat repository GitHub
  1. Buka github.com → Login atau daftar dulu
  2. Klik tombol New (hijau) di pojok kiri atas
  3. Nama repository: gym-tracker
  4. Pilih Public (biar bisa diakses dari mana saja)
  5. Centang Add a README file
  6. Klik Create repository
Langkah 2 — Upload file HTML ini
  1. Di halaman repository yang baru dibuat, klik Add file → Upload files
  2. Upload file gym-tracker.html ini
  3. Rename file menjadi index.html (penting! supaya jadi halaman utama)
  4. Scroll bawah, klik Commit changes
Langkah 3 — Aktifkan GitHub Pages
  1. Di repository, klik tab Settings
  2. Scroll ke bagian Pages di sidebar kiri
  3. Di bagian Source, pilih Deploy from a branch
  4. Branch: main, folder: / (root)
  5. Klik Save
  6. Tunggu 1–2 menit, refresh halaman
  7. URL akan muncul: https://username.github.io/gym-tracker
Cara pakai dari HP / laptop manapun (tanpa kehilangan data)

Karena ini pure HTML, data tidak otomatis sync antar device. Alurnya simpel:

  1. Buka tracker di device yang kamu pakai terakhir
  2. Klik Export JSON → simpan file gym-data.json
  3. Upload gym-data.json ke repository GitHub (di folder yang sama dengan index.html)
  4. Di device baru, buka tracker → klik Import JSON → pilih file tersebut

Tip: Simpan data JSON di Google Drive atau WhatsApp ke diri sendiri, lalu import di device manapun. Data tidak akan hilang selama file JSON-nya ada.

Bookmark URL-nya di HP
Setelah GitHub Pages aktif, buka URL-nya di Chrome/Safari → tap menu browser → Add to Home Screen. Tracker akan muncul seperti app di home screen HP kamu, bisa dibuka kapan saja tanpa install apapun.
